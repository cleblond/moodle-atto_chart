Moodle 2.7 Atto plugin for inserting dhtmlxCharts in moodle.
Requires moodle-filter_charts plugin.
Copyright 2014 onwards Carl LeBlond


Allows inserting of GPL dhtmlxCharts (http://dhtmlx.com/docs/products/dhtmlxSpreadsheet/)
-Create editable or readonly grids/charts
-Groups editing based on course group membership
